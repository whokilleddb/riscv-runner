# RISC-V Runner

A RISC-V emulator written in Rust inspired by the [RVEmu Book](https://book.rvemu.app/index.html).

## Pre-requisites

- [riscv-gnu-toolchain](https://github.com/riscv-collab/riscv-gnu-toolchain)
- cargo
